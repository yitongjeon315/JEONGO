# 🤖 AI 미니 문서 스캐너 (AI Document Scanner)

이 프로젝트는 Node.js와 Express, 그리고 Tesseract.js를 사용하여 만든 아주 간단하고 직관적인 **웹 기반 문서 스캐너**입니다. 
초보자도 쉽게 실행하고 테스트할 수 있도록 직관적인 UI와 쉬운 실행 구조로 제작되었습니다.

---

## ✨ 주요 기능

1. **📁 이미지 업로드 및 실시간 미리보기**
   - 파일 선택창을 통한 업로드뿐만 아니라 **드래그 앤 드롭(Drag & Drop)** 방식의 편리한 이미지 업로드를 지원합니다.
   - 이미지를 선택하는 즉시 화면에 실시간으로 미리보기가 나타납니다.

2. **🔍 한글 & 영어 문자 인식 (OCR)**
   - 오픈소스 OCR 엔진인 `Tesseract.js`를 연동하여 이미지 속 한글과 영어 텍스트를 정밀하게 추출합니다.

3. **🤖 규칙 기반 AI 문서 분석 및 요약**
   - 추출된 텍스트 데이터를 분석하여 문서의 유형을 자동으로 추정하고 한 줄 요약을 제공합니다.
     - **영수증**: '원', '합계', '카드', '현금' 등의 단어가 감지될 때
     - **명함/연락처**: '@', '010', 'Tel', '전화' 등의 단어가 감지될 때
     - **안내문**: '공지', '안내', '기간', '장소' 등의 단어가 감지될 때

4. **📋 원클릭 복사 기능**
   - 추출된 텍스트 전체를 원클릭으로 클립보드에 복사할 수 있습니다.

5. **💾 TXT 파일 결과 저장**
   - 스캔한 파일명, 분석 날짜, 문서 종류, 한 줄 요약, 그리고 OCR로 추출된 전체 텍스트를 포함한 보고서를 `scan_result_날짜시간.txt` 파일로 내 컴퓨터에 바로 저장할 수 있습니다.

## 🚀 실행 방법

### 1. 프로젝트 폴더로 이동
터미널(명령 프롬프트 또는 파워쉘)을 열고 프로젝트가 설치된 폴더로 이동합니다.
```bash
cd "%USERPROFILE%\Desktop\AI_문서스캐너_7주차"
```

### 2. 패키지 설치
프로젝트에 필요한 패키지들(Express, Multer, Tesseract.js)을 설치합니다. (이미 완료된 경우 생략 가능)
```bash
npm install
```

### 3. 서버 실행
아래 명령어를 입력하여 로컬 웹 서버를 구동합니다.
```bash
node server.js
```

### 4. 웹 브라우저 접속
서버가 켜지면 브라우저 주소창에 아래 주소를 입력하여 접속합니다.
👉 **[http://localhost:5001](http://localhost:5001)**
